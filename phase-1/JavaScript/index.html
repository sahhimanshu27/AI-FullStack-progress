How to View JavaScript Output (console.log)

You can view JavaScript logs and script outputs using the browser's Developer Console. Use the shortcuts below:

Chrome / Edge / Brave / Opera

  • Open DevTools: Ctrl + Shift + I (Windows) / Cmd + Option + I (Mac)
  • Open Console Directly: Ctrl + Shift + J (Windows) / Cmd + Option + J (Mac)

Firefox

  • Open DevTools: Ctrl + Shift + I / Cmd + Option + I
  • Open Web Console: Ctrl + Shift + K / Cmd + Option + K

Safari (Mac)

First enable the Developer Menu in Safari:
Safari → Preferences → Advanced → check “Show Develop menu in menu bar”.

  • Open Console: Cmd + Option + C
  • Open DevTools: Cmd + Option + I

Where Do JS Logs Appear?

After opening DevTools, click the Console tab. That’s where all console.log() outputs appear.

3. Type Conversion.

Demonstrating type conversion in JavaScript.


// String Conversion
let value = true;
console.log(typeof value); // boolean
value = String(value); // now value is a string "true"
console.log(typeof value); // string

// Numeric Conversion
let str = "123";
console.log(typeof str); // string
let num = Number(str); // becomes the number 123
console.log(typeof num); // number

let ageCon = Number("an arbitrary string instead of a number");
console.log(ageCon); // NaN, conversion failed

// Boolean Conversion
console.log(Boolean(1)); // true
console.log(Boolean(0)); // false

console.log(Boolean("hello")); // true
console.log(Boolean("")); // false
